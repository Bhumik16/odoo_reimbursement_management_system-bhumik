# ⚙️ Setup Instructions

## 1. Clone the Repository

```
git clone <your-repo-url>
cd reimbursehq
```

---

## 2. Install Dependencies

### Client

```
cd client
npm install
```

### Server

```
cd server
npm install
```

---



## 3. Run the Application

### Start Backend

```
cd server
node server.js
```

### Start Frontend

```
cd client
npm run dev
```
